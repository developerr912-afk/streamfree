# StreamFree

A modern movie streaming web application built using HTML, CSS, JavaScript, and Firebase.

## Features

- User Authentication
- Responsive Design
- Movie Browsing
- Search Functionality
- Modern UI

## Tech Stack

- HTML
- CSS
- JavaScript
- Firebase

## Installation

1. Clone the repository
2. Open the project folder
3. Run using Live Server or open index.html
